Font Properties

Font Family

Sets the typeface for text content.

.element { font-family: 'Arial', sans-serif; font-family: 'Times New Roman', serif; font-family: 'Courier New', monospace; }

Font Size

Controls the size of text characters.

.element { font-size: 16px; /* Pixels */ font-size: 1.2em; /* Relative to parent */ font-size: 1.5rem; /* Relative to root */ font-size: large; /* Keyword */ }

12px - Small text

16px - Normal text

24px - Large text

32px - Extra large

Font Weight

Controls the thickness/boldness of characters.

.element { font-weight: normal; /* 400 */ font-weight: bold; /* 700 */ font-weight: 300; /* Light */ font-weight: 900; /* Extra Bold */ }

400 - Normal

600 - Semi Bold

700 - Bold

Google Fonts Integration

How to Use Google Fonts

Google Fonts provides free, high-quality web fonts that you can easily integrate into your website.

Step 1: Choose Your Fonts

Visit fonts.google.com and select your desired fonts.

Step 2: Add to HTML

<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap" rel="stylesheet">

Step 3: Use in CSS

body { font-family: 'Inter', sans-serif; }

Font Examples:

Inter - Modern and clean

Playfair Display - Elegant serif

Source Code Pro - Monospace
